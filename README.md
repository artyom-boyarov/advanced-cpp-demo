# Advanced C++ Demo
Demonstration code covering templates, enums, and smart pointers.

## Build and run
```
mkdir build/
cd build/
cmake ..
make -j4
```
Then run `./enums`, `./templates`, or `./smartptrs`

## Build and run - CMake Demo
```
cd cmake_demo/
mkdir build/
cd build/
cmake ..
make -j4
./main_exec
```
